会場の感染拡大防止措置に基づき、第4回演奏会での感染防止チェックリストを以下に公表します。

![](blog/fourth-concert-checklist/images/checklist-1.jpg)
![](blog/fourth-concert-checklist/images/checklist-2.jpg)

[出典：感染防止策チェックリスト【第1版（令和3年11月版）】](http://www.city.hiratsuka.kanagawa.jp/common/200098115.pdf)

<!-- ### 1. 飛沫の抑制（マスク着用や大声を出さないこと）の徹底 -->

<!-- - [x]  飛沫が発生するおそれのある行為を抑制するため、適切なマスク（品質の確かな、できれば不織布）の正しい着用や大声を出さないことを周知・徹底し、そうした行為をする者がいた場合には、個別に注意、退場処分等の措置を講じる。 -->
<!-- ※大声の定義を「観客等が、(1)通常よりも大きな声量で、(2)反復・継続的に声を発すること」とする。 -->

<!-- ### 2. 手洗い、手指・設備消毒の徹底 -->

<!-- - [x]  こまめな手洗や手指消毒の徹底を促す（会場出入口等へのアルコール等の手指消毒液の設置や場内アナウンス等の実施） -->
<!-- - [x]  主催者側による施設内（出入口、共用部等）の定期的かつこまめな消毒の実施 -->

<!-- ### 3. 換気の徹底 -->

<!-- - [x]  法令を遵守した空調設備の設置による常時換気又はこまめな換気（1時間に2回以上・1回に5分間以上等）の徹底 -->

<!-- ### 4. 来場者間の密集回避 -->

<!-- - [x]  入退場時の密集を回避するための措置（入場ゲートの増設や時間差入退場等）の実施 -->
<!-- - [x]  休憩時間や待合場所での密集も回避するための人員配置や導線確保等の体制構築 -->
<!-- - [x]  大声を伴わない場合には、人と人とが触れ合わない間隔、大声を伴う可能性のあるイベントは、前後左右の座席と身体的距離の確保 -->

<!-- ### 5. 飲食の制限（飲食は楽屋またはホワイエに限る） -->

<!-- - [x]  飲食時の感染防止策（飲食店に求められる感染防止策等を踏まえた十分な対策）の徹底 -->
<!-- - [x]  飲食中以外のマスク着用の推奨 -->
<!-- - [x]  飲食は、飛沫感染のリスクを高めるため、楽屋またはホワイエ以外ではしない -->
<!-- - [x]  主催者、参加者とも飲食はしない -->

<!-- ### 6. 出演者等の感染対策 -->

<!-- - [x]  有症状者（発熱又は風邪等の症状を呈する者）は出演・練習を控えるなど日常から出演者やスタッフ等の健康管理を徹底する -->
<!-- - [x]  練習時等、イベント開催前も含め、声を発出する出演者やスタッフ等の関係者間での感染リスクに対処する -->
<!-- - [x]  出演者やスタッフ等と観客がイベント前後・休憩時間等に接触しないよう確実な措置を講じる（誘導スタッフ等必要な場合を除く） -->

<!-- ### 7. 参加者の把握・管理等 -->

<!-- - [x]  入場者（参加者）確定時又は入場時の連絡先確認やアプリ等を活用した参加者の把握 -->
<!-- - [x]  入場時の検温、有症状者（発熱又は風邪等の症状）等を理由に入場できなかった際の払い戻し措置等により、有症状者の入場を確実に防止 -->
<!-- - [x]  時差入退場の実施や直行・直帰の呼びかけ等イベント前後の感染防止の注意喚起 -->
